# PI1-Jogo-SimonSays-Arduino
Projeto Integrador desenvolvido na FATEC Santo André por Gustavo Felipe Dias, o Simon Says utiliza Arduino Uno e C++ para recriar o clássico Genius. O sistema integra lógica sequencial e eletrônica básica para gerenciar sequências aleatórias, leitura de botões e feedbacks de LEDs e buzzer, demonstrando domínio em manipulação de I/O e hardware.
